## Important Remarks / Prerequisites

1. The script provided in run_analysis.R is a function called run_analysis(). Please compile the function and run it. 
2. The working directory needs to include the extracted source data in its folder structure. The top level folder is called 'UCI HAR Dataset'
3. The script requires the package 'dplyr'
4. The result will be returned in the following file in the working directory: TidyMeans.txt
